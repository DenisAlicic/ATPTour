Packs a tarball from a local directory or from a registry or github spec and returns a Promise that resolves to the tarball data Buffer, with from, resolved, and integrity fields attached.

If no options are passed, the tarball file will be saved on the same directory from which {\ttfamily pack} was called in.

{\ttfamily libnpmpack} uses \href{https://npm.im/pacote}{\texttt{ {\ttfamily pacote}}}. Most options are passed through directly to that library, so please refer to \href{https://www.npmjs.com/package/pacote\#options}{\texttt{ its own {\ttfamily opts} documentation}} for options that can be passed in.\hypertarget{md__home_denis_Documents_Faks_master_RS2_TennisAssociation_TennisAssociation_Client_node_modulesd51e302372d2e106b48a393d051ad2d0_autotoc_md13821}{}\doxyparagraph{Examples}\label{md__home_denis_Documents_Faks_master_RS2_TennisAssociation_TennisAssociation_Client_node_modulesd51e302372d2e106b48a393d051ad2d0_autotoc_md13821}
