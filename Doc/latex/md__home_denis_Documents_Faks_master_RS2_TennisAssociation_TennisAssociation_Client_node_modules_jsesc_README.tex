Given some data, {\itshape jsesc} returns a stringified representation of that data. jsesc is similar to {\ttfamily JSON.\+stringify()} except\+:

\item its output is ASCII-\/safe \href{\#minimal}{\texttt{ by default}}, thanks to its use of \href{https://mathiasbynens.be/notes/javascript-escapes}{\texttt{ escape sequences}} where needed.
\end{DoxyEnumerate}
