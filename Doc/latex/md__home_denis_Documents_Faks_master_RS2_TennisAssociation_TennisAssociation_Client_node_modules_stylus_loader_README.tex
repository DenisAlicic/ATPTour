{\ttfamily stylus-\/loader} currently prefers resolving paths with stylus\textquotesingle{}s resovling utilities and then falling back to webpack when it can\textquotesingle{}t find files. Use the {\ttfamily prefer\+Path\+Resolver} option with the value `\textquotesingle{}webpack'\`{} to swap this. This has the benefit of using webpack\textquotesingle{}s async resolving instead of stylus\textquotesingle{}s sync resolving. If you have a lot of dependencies in your stylus files this\textquotesingle{}ll let those dependencies be found in parallel.
