Compare \href{https://semver.org/}{\texttt{ semver}} version strings to find greater, equal or lesser. Runs in the browser as well as Node.\+js/\+React Native etc. Has no dependencies and is tiny ($\sim$630 bytes gzipped).

This library supports the full semver specification, including comparing versions with different number of digits like {\ttfamily 1.\+0.\+0}, {\ttfamily 1.\+0}, {\ttfamily 1}, and pre-\/release versions like {\ttfamily 1.\+0.\+0-\/alpha}. Additionally supports the following variations\+:


\begin{DoxyItemize}
\item Supports wildcards for minor and patch version like {\ttfamily 1.\+0.\+x} or {\ttfamily 1.\+0.$\ast$}.
\item Supports \href{https://www.chromium.org/developers/version-numbers}{\texttt{ Chromium version numbers}} with 4 parts, e.\+g. version {\ttfamily 25.\+0.\+1364.\+126}.
\item Any leading {\ttfamily v} is ignored, e.\+g. {\ttfamily v1.\+0} is interpreted as {\ttfamily 1.\+0}.
\item Leading zero is ignored, e.\+g. {\ttfamily 1.\+01.\+1} is interpreted as {\ttfamily 1.\+1.\+1}.
