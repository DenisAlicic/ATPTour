{\ttfamily \textbackslash{}} is considered a separator only on WIN32 systems. All POSIX compliant systems see backslash as a valid file name character, so it would break POSIX compliance to remove it there.

In practice, this means that this code will return different things depending on what system it runs on\+:


\begin{DoxyCode}{0}
\DoxyCodeLine{removeTrailingSeparator('\(\backslash\)\(\backslash\)foo\(\backslash\)\(\backslash\)')}
\DoxyCodeLine{// UNIX  => '\(\backslash\)\(\backslash\)foo\(\backslash\)\(\backslash\)'}
\DoxyCodeLine{// WIN32 => '\(\backslash\)\(\backslash\)foo'}

\end{DoxyCode}
