Create a pipeline of streams using Minipass.

Calls {\ttfamily .pipe()} on all the streams in the list. Returns a stream where writes got to the first pipe in the chain, and reads are from the last.

Errors are proxied along the chain and emitted on the Pipeline stream.\hypertarget{md__home_denis_Documents_Faks_master_RS2_TennisAssociation_TennisAssociation_Client_node_modules40edafa56c6a97d2a5605861ce24b8ba_autotoc_md13950}{}\doxysection{USAGE}\label{md__home_denis_Documents_Faks_master_RS2_TennisAssociation_TennisAssociation_Client_node_modules40edafa56c6a97d2a5605861ce24b8ba_autotoc_md13950}
