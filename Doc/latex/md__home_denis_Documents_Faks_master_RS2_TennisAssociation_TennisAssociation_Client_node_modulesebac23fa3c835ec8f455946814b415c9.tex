Checks if there are any conflicting bins which will prevent the linking of bins for the given package. Returns a Promise that resolves with no value if the way is clear, and rejects if there\textquotesingle{}s something in the way.

Always returns successfully if {\ttfamily global} or {\ttfamily top} are false, or if {\ttfamily force} is true, or if the {\ttfamily pkg} object does not contain any bins to link.

Note that changes to the file system {\itshape may} still cause the {\ttfamily bin\+Links} method to fail even if this method succeeds. Does not check for conflicting {\ttfamily man} links.

Reads from the filesystem but does not make any changes.\hypertarget{md__home_denis_Documents_Faks_master_RS2_TennisAssociation_TennisAssociation_Client_node_modulesebac23fa3c835ec8f455946814b415c9_autotoc_md13700}{}\doxyparagraph{Example}\label{md__home_denis_Documents_Faks_master_RS2_TennisAssociation_TennisAssociation_Client_node_modulesebac23fa3c835ec8f455946814b415c9_autotoc_md13700}
