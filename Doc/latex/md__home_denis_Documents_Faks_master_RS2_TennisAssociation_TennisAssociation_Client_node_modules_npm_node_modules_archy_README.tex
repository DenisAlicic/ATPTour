Return a string representation of {\ttfamily obj} with unicode pipe characters like how {\ttfamily npm ls} looks.

{\ttfamily obj} should be a tree of nested objects with `\textquotesingle{}label'{\ttfamily and}\textquotesingle{}nodes\textquotesingle{}{\ttfamily fields. }\textquotesingle{}label\textquotesingle{}{\ttfamily is a string of text to display at a node level and}\textquotesingle{}nodes\textquotesingle{}\`{} is an array of the descendents of the current node.

If a node is a string, that string will be used as the `\textquotesingle{}label'{\ttfamily and an empty array of}\textquotesingle{}nodes\textquotesingle{}\`{} will be used.

{\ttfamily prefix} gets prepended to all the lines and is used by the algorithm to recursively update.

If `\textquotesingle{}label'\`{} has newlines they will be indented at the present indentation level with the current prefix.
