The {\ttfamily killable} module is callable. When you call it on a Node {\ttfamily http.\+Server} object, it will add a {\ttfamily server.\+kill()} method on it. It returns the server object.

{\ttfamily server.\+kill(\mbox{[}callback\mbox{]})} closes all open sockets and calls {\ttfamily server.\+close()}, to which the {\ttfamily callback} is passed on.
