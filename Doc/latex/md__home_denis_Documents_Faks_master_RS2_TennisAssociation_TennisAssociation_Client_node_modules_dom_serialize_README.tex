For custom serialization logic, a \char`\"{}serialize\char`\"{} event is dispatched on every {\ttfamily Node} which event listeners can override the default behavior on by setting the {\ttfamily event.\+detail.\+serialize} property to a String or other Node.

The \char`\"{}serialize\char`\"{} event bubbles, so it could be a good idea to utilize event delegation on a known root node that will be serialized.
