Fetches the registry tarballs and compare files between a spec {\ttfamily a} and spec {\ttfamily b}. {\bfseries{npm}} spec types are usually described in {\ttfamily $<$pkg-\/name$>$@$<$version$>$} form but multiple other types are alsos supported, for more info on valid specs take a look at \href{https://github.com/npm/npm-package-arg}{\texttt{ {\ttfamily npm-\/package-\/arg}}}.

{\bfseries{Options}}\+:


\begin{DoxyItemize}
\item {\ttfamily color $<$Boolean$>$}\+: Should add ANSI colors to string output? Defaults to {\ttfamily false}.
\item {\ttfamily tag\+Version\+Prefix $<$Sring$>$}\+: What prefix should be used to define version numbers. Defaults to {\ttfamily v}
\item {\ttfamily diff\+Unified $<$Number$>$}\+: How many lines of code to print before/after each diff. Defaults to {\ttfamily 3}.
\item {\ttfamily diff\+Files $<$Array$<$String$>$$>$}\+: If set only prints patches for the files listed in this array (also accepts globs). Defaults to {\ttfamily undefined}.
\item {\ttfamily diff\+Ignore\+All\+Space $<$Boolean$>$}\+: Whether or not should ignore changes in whitespace (very useful to avoid indentation changes extra diff lines). Defaults to {\ttfamily false}.
\item {\ttfamily diff\+Name\+Only $<$Boolean$>$}\+: Prints only file names and no patch diffs. Defaults to {\ttfamily false}.
\item {\ttfamily diff\+No\+Prefix $<$Boolean$>$}\+: If true then skips printing any prefixes in filenames. Defaults to {\ttfamily false}.
\item {\ttfamily diff\+Src\+Prefix $<$String$>$}\+: Prefix to be used in the filenames from {\ttfamily a}. Defaults to {\ttfamily a/}.
\item {\ttfamily diff\+Dst\+Prefix $<$String$>$}\+: Prefix to be used in the filenames from {\ttfamily b}. Defaults to {\ttfamily b/}.
\item {\ttfamily diff\+Text $<$Boolean$>$}\+: Should treat all files as text and try to print diff for binary files. Defaults to {\ttfamily false}.
\item ...{\ttfamily cache}, {\ttfamily registry}, {\ttfamily where} and other common options accepted by \href{https://github.com/npm/pacote\#options}{\texttt{ pacote}}
\end{DoxyItemize}

Returns a {\ttfamily Promise} that fullfils with a {\ttfamily String} containing the resulting patch diffs.
