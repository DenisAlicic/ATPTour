Supports four methods for every class\+:


\begin{DoxyItemize}
\item encode — encodes, replacing characters to its entity representations. Ignores UTF characters with no entity representation.
\item encode\+Non\+UTF — encodes, replacing characters to its entity representations. Inserts numeric entities for UTF characters.
\item encode\+Non\+ASCII — encodes, replacing only non-\/\+ASCII characters to its numeric entity representations.
\item decode — decodes, replacing entities to characters. Unknown entities are left as is. 
\end{DoxyItemize}
