This project is an ISO 3166 (Country codes) module for Angular\+JS. It provides\+:


\begin{DoxyItemize}
\item A service (factory) that gives access to all country codes
\item A filter to print country codes as their standard name (FR -\/$>$ FRANCE)
\item A validation directive to validate country codes
\end{DoxyItemize}\hypertarget{md__home_denis_Documents_Faks_master_RS2_TennisAssociation_TennisAssociation_Client_node_modulesad5f987fe69cae7da4f63809196d5efa_autotoc_md31538}{}\doxysubsection{Get it}\label{md__home_denis_Documents_Faks_master_RS2_TennisAssociation_TennisAssociation_Client_node_modulesad5f987fe69cae7da4f63809196d5efa_autotoc_md31538}
This library is available with the bower package manager, you can either\+:


\begin{DoxyItemize}
\item Execute the following command\+: {\ttfamily bower install iso-\/3166-\/country-\/codes-\/angular -\/-\/save}
\item Add this line in your dependencies\+: {\ttfamily \char`\"{}iso-\/3166-\/country-\/codes-\/angular\char`\"{}\+: \char`\"{}1.\+1.\+1\char`\"{}}
\end{DoxyItemize}\hypertarget{md__home_denis_Documents_Faks_master_RS2_TennisAssociation_TennisAssociation_Client_node_modulesad5f987fe69cae7da4f63809196d5efa_autotoc_md31539}{}\doxysubsection{Features}\label{md__home_denis_Documents_Faks_master_RS2_TennisAssociation_TennisAssociation_Client_node_modulesad5f987fe69cae7da4f63809196d5efa_autotoc_md31539}

If you get country codes from your REST server, you can print their standard names with the provided filter\+:


\begin{DoxyCode}{0}
\DoxyCodeLine{<!-\/-\/ if countryCode is 'FR', will print 'France' -\/-\/>}
\DoxyCodeLine{<p>\{\{countryCode | isoCountry \}\}</p>}

\end{DoxyCode}


On the other hand, if you get country names from your REST server, you can print their ISO codes\+:


\begin{DoxyCode}{0}
\DoxyCodeLine{<!-\/-\/ if countryName is 'France', will print 'FR' -\/-\/>}
\DoxyCodeLine{<p>\{\{countryName | isoCountryCode \}\}</p>}

\end{DoxyCode}
