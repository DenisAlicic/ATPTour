Codelyzer supports any template and style language by custom hooks. If you\textquotesingle{}re using Sass for instance, you can allow codelyzer to analyze your styles by creating a file {\ttfamily .codelyzer.\+js} in the root of your project (where the {\ttfamily node\+\_\+modules} directory is). In the configuration file can implement custom pre-\/processing and template resolution logic\+:
