All patterns can be divided into two types\+:


\begin{DoxyItemize}
\item {\bfseries{static}}. A pattern is considered static if it can be used to get an entry on the file system without using matching mechanisms. For example, the {\ttfamily file.\+js} pattern is a static pattern because we can just verify that it exists on the file system.
\item {\bfseries{dynamic}}. A pattern is considered dynamic if it cannot be used directly to find occurrences without using a matching mechanisms. For example, the {\ttfamily $\ast$} pattern is a dynamic pattern because we cannot use this pattern directly.
\end{DoxyItemize}

A pattern is considered dynamic if it contains the following characters ({\ttfamily …} — any characters or their absence) or options\+:


\begin{DoxyItemize}
\item The \href{\#casesensitivematch}{\texttt{ {\ttfamily case\+Sensitive\+Match}}} option is disabled
\item {\ttfamily \textbackslash{}\textbackslash{}} (the escape character)
\item {\ttfamily $\ast$}, {\ttfamily ?}, {\ttfamily !} (at the beginning of line)
\item {\ttfamily \mbox{[}…\mbox{]}}
\item {\ttfamily (…$\vert$…)}
\item {\ttfamily @(…)}, {\ttfamily !(…)}, {\ttfamily $\ast$(…)}, {\ttfamily ?(…)}, {\ttfamily +(…)} (respects the \href{\#extglob}{\texttt{ {\ttfamily extglob}}} option)
