\item The {\ttfamily karma} module does not export {\ttfamily karma} binary anymore. The recommended way is to have local modules (karma and all the plugins that your project needs) stored in your {\ttfamily package.\+json}. You can run that particular Karma by {\ttfamily ./node\+\_\+modules/karma/bin/karma}. Or you can have {\ttfamily karma-\/cli} installed globally on your system, which enables you to use the {\ttfamily karma} command.
\end{DoxyItemize}

The global {\ttfamily karma} command (installed by {\ttfamily karma-\/cli}) does look for local version of Karma (including parent directories) first and fall backs to a global one.

The {\ttfamily bin/karma} binary does not look for any other instances of Karma and just runs the one that it belongs to.

(\href{http://github.com/karma-runner/karma/commit/c7d46270aca83ecfe78f69fa923bc574c0b5bfdc}{\texttt{ c7d46270}})
