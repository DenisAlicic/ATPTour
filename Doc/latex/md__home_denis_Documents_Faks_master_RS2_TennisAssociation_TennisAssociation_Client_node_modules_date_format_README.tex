Format string can be anything, but the following letters will be replaced (and leading zeroes added if necessary)\+:
\begin{DoxyItemize}
\item dd -\/ {\ttfamily date.\+get\+Date()}
\item MM -\/ {\ttfamily date.\+get\+Month() + 1}
\item yy -\/ {\ttfamily date.\+get\+Full\+Year().to\+String().substring(2, 4)}
\item yyyy -\/ {\ttfamily date.\+get\+Full\+Year()}
\item hh -\/ {\ttfamily date.\+get\+Hours()}
\item mm -\/ {\ttfamily date.\+get\+Minutes()}
\item ss -\/ {\ttfamily date.\+get\+Seconds()}
\item SSS -\/ {\ttfamily date.\+get\+Milliseconds()}
\item O -\/ timezone offset in +hm format (note that time will be in UTC if displaying offset)
\end{DoxyItemize}

The date format library has limited ability to parse strings into dates. It can convert strings created using date format patterns (as above), but if you\textquotesingle{}re looking for anything more sophisticated than that you should probably look for a better library (\href{https://momentjs.com}{\texttt{ momentjs}} does pretty much everything).


\begin{DoxyCode}{0}

\DoxyCodeLine{// pass in the format of the string as first argument}
\DoxyCodeLine{format.parse(format.ISO8601\_FORMAT, '2017-\/03-\/14T14:10:20.391');}
\DoxyCodeLine{// returns Date}

\end{DoxyCode}
