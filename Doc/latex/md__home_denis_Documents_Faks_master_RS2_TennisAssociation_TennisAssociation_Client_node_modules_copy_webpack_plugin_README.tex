The {\ttfamily context} option can be an absolute or relative path. If {\ttfamily context} is a relative, then it is converted to absolute based to {\ttfamily compiler.\+options.\+context}

Also, {\ttfamily context} indicates how to interpret the search results. Further, he is considered in this role.

To determine the structure from which the found resources will be copied to the destination folder, the {\ttfamily context} option is used.

If {\ttfamily from} is a file, then {\ttfamily context} is equal to the directory in which this file is located. Accordingly, the result will be only the file name.

If {\ttfamily from} is a directory, then {\ttfamily context} is the same as {\ttfamily from} and is equal to the directory itself. In this case, the result will be a hierarchical structure of the found folders and files relative to the specified directory.

If {\ttfamily from} is a glob, then regardless of the {\ttfamily context} option, the result will be the structure specified in the {\ttfamily from} option
