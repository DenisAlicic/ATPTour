Instantiates an options object defined by {\ttfamily figgy\+Pudding()}, which uses {\ttfamily providers}, in order, to find requested properties.

Each provider can be either a plain object, a {\ttfamily Map}-\/like object (that is, one with a {\ttfamily .get()} method) or another figgy\+Pudding {\ttfamily Opts} object.

When nesting {\ttfamily Opts} objects, their properties will not become available to the new object, but any further nested {\ttfamily Opts} that reference that property {\itshape will} be able to read from their grandparent, as long as they define that key. Default values for nested {\ttfamily Opts} parents will be used, if found.
