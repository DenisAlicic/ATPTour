In Node.\+js, lie emits an {\ttfamily unhandled\+Rejection} event when a rejected promise isn\textquotesingle{}t caught, in line with \href{https://iojs.org/api/process.html\#process_event_unhandledrejection}{\texttt{ how io.\+js does it}}. This allows it to act as a promise shim in both Node.\+js and the browser.
