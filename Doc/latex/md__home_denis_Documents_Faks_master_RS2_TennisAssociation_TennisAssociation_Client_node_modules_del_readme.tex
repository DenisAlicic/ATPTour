\begin{quote}
Delete files and folders using \href{https://github.com/isaacs/minimatch\#usage}{\texttt{ globs}} \end{quote}
Similar to \href{https://github.com/isaacs/rimraf}{\texttt{ rimraf}}, but with a Promise API and support for multiple files and globbing. It also protects you against deleting the current working directory and above.
