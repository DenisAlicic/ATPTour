\doxysubsection*{Context}

When using a {\ttfamily \{Function\}} ({\ttfamily postcss.\+config.\+js} or {\ttfamily .postcssrc.\+js}), it\textquotesingle{}s possible to pass context to {\ttfamily postcss-\/load-\/config}, which will be evaluated while loading your config. By default {\ttfamily ctx.\+env (process.\+env.\+NODE\+\_\+\+ENV)} and {\ttfamily ctx.\+cwd (\mbox{\hyperlink{namespaceupdate-gyp_a353f325a85a850b0456a7cbaff8fc43b}{process.\+cwd()}})} are available on the {\ttfamily ctx} {\ttfamily \{Object\}}

\begin{quote}
ℹ️ Most third-\/party integrations add additional properties to the {\ttfamily ctx} (e.\+g {\ttfamily postcss-\/loader}). Check the specific module\textquotesingle{}s README for more information about what is available on the respective {\ttfamily ctx} \end{quote}
