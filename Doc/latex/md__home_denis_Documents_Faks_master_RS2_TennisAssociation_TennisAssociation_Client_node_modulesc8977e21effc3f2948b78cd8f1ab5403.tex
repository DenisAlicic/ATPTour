SMP has 2 primary parts\+:


\begin{DoxyEnumerate}
\item The \href{./index.js}{\texttt{ SMP class}} contains the {\ttfamily smp.\+wrap} instance method that bootstraps the whole wrapping sequence. This class also listens for basic timing events, and orchestrates the main timings.
\item The \href{./WrappedPlugin}{\texttt{ {\ttfamily Wrapped\+Plugin}}} proxy that wraps each webpack plugin. This uses a {\ttfamily Proxy} to wrap everything to do with a plugin, feeding the timing information back to the SMP class. 
\end{DoxyEnumerate}
