\begin{quote}
Color your terminal using pure idiomatic Java\+Script. \end{quote}
Colorette is a Node.\+js library for embellishing your CLI tools with colors and styles using \href{https://en.wikipedia.org/wiki/ANSI_escape_code}{\texttt{ ANSI escape codes}}.


\begin{DoxyItemize}
\item $\sim$1.5x faster than alternatives (\href{\#run-the-benchmarks}{\texttt{ run the benchmarks}}).
\item No wonky prototype-\/based method chains.
\item Automatic color support detection.
\item $\sim$80 LOC and no dependencies.
\item \href{https://no-color.org}{\texttt{ {\ttfamily NO\+\_\+\+COLOR}}} friendly.
\end{DoxyItemize}\hypertarget{md__home_denis_Documents_Faks_master_RS2_TennisAssociation_TennisAssociation_Client_node_modules_colorette_README_autotoc_md22508}{}\doxysection{Quickstart}\label{md__home_denis_Documents_Faks_master_RS2_TennisAssociation_TennisAssociation_Client_node_modules_colorette_README_autotoc_md22508}

\begin{DoxyCode}{0}
\DoxyCodeLine{npm i colorette}

\end{DoxyCode}


Import the \href{\#styles}{\texttt{ styles}} you need. \href{\#supported-styles}{\texttt{ Here}}\textquotesingle{}s the list of styles you can use.


\begin{DoxyCode}{0}
\DoxyCodeLine{import \{ red, blue, bold \} from "{}colorette"{}}

\end{DoxyCode}


Wrap your strings in one or more styles to produce the finish you\textquotesingle{}re looking for.


\begin{DoxyCode}{0}
\DoxyCodeLine{console.log(bold(blue("{}Engage!"{})))}

\end{DoxyCode}


Mix it with \href{https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference/Template_literals}{\texttt{ template literals}} to interpolate variables, expressions and create multi-\/line strings easily.


\begin{DoxyCode}{0}
\DoxyCodeLine{console.log(`}
\DoxyCodeLine{  Beets are \$\{red("{}red"{})\},}
\DoxyCodeLine{  Plums are \$\{blue("{}blue"{})\},}
\DoxyCodeLine{  \$\{bold("{}Colorette!"{})\}.}
\DoxyCodeLine{`)}

\end{DoxyCode}


Using {\ttfamily console.\+log}\textquotesingle{}s \href{https://nodejs.org/api/console.html\#console_console_log_data_args}{\texttt{ string substitution}} can be useful too.


\begin{DoxyCode}{0}
\DoxyCodeLine{console.log(bold("{}Total: \$\%f"{}), 1.99)}

\end{DoxyCode}


You can also nest styles without breaking existing escape codes.


\begin{DoxyCode}{0}
\DoxyCodeLine{console.log(red(`Red Shirt \$\{blue("{}Blue Shirt"{})\} Red Shirt`))}

\end{DoxyCode}


Feeling adventurous? Try the \href{https://github.com/tc39/proposal-pipeline-operator}{\texttt{ pipeline operator}}.


\begin{DoxyCode}{0}
\DoxyCodeLine{console.log("{}Make it so!"{} |> bold |> blue)}

\end{DoxyCode}
