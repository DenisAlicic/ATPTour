Keep in mind that the target object will be modified, and will be returned from extend().

If a boolean true is specified as the first argument, extend performs a deep copy, recursively copying any objects it finds. Otherwise, the copy will share structure with the original object(s). Undefined properties are not copied. However, properties inherited from the object\textquotesingle{}s prototype will be copied over. Warning\+: passing {\ttfamily false} as the first argument is not supported.\hypertarget{md__home_denis_Documents_Faks_master_RS2_TennisAssociation_TennisAssociation_Client_node_modules_npm_node_modules_extend_README_autotoc_md16334}{}\doxysubsubsection{Arguments}\label{md__home_denis_Documents_Faks_master_RS2_TennisAssociation_TennisAssociation_Client_node_modules_npm_node_modules_extend_README_autotoc_md16334}
