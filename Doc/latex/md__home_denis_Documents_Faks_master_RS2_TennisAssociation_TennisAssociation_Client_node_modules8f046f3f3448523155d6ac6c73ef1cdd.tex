This utility allows you to render unicode-\/aided tables on the command line from your node.\+js scripts.

{\ttfamily cli-\/table3} is based on (and api compatible with) the original \href{https://github.com/Automattic/cli-table}{\texttt{ cli-\/table}}, and \href{https://github.com/jamestalmage/cli-table2}{\texttt{ cli-\/table2}}, which are both unmaintained. {\ttfamily cli-\/table3} includes all the additional features from {\ttfamily cli-\/table2}.

\begin{DoxyItemize}
\item Ability to make cells span columns and/or rows.
\item Ability to set custom styles per cell (border characters/colors, padding, etc).
\item Vertical alignment (top, bottom, center).
\item Automatic word wrapping.
\item More robust truncation of cell text that contains ansi color characters.
\item Better handling of text color that spans multiple lines.
\item API compatible with the original cli-\/table.
\item Exhaustive test suite including the entire original cli-\/table test suite.
\item Lots of examples auto-\/generated from the tests (\href{https://github.com/cli-table/cli-table3/blob/master/basic-usage.md}{\texttt{ basic}}, \href{https://github.com/cli-table/cli-table3/blob/master/advanced-usage.md}{\texttt{ advanced}}).

Cross tables are very similar to vertical tables, with two key differences\+:


\begin{DoxyEnumerate}
\item They require a {\ttfamily head} setting when instantiated that has an empty string as the first header
\item The individual rows take the general form of \{ \char`\"{}\+Header\char`\"{}\+: \mbox{[}\char`\"{}\+Row\char`\"{}, \char`\"{}\+Values\char`\"{}\mbox{]} \}
\end{DoxyEnumerate}
