The module returns the address of the internet-\/facing interface, as determined from the default gateway. When the address cannot be determined for any reason, {\ttfamily null} will be returned.

The module relies on operating systems tools. On Linux and Android, the {\ttfamily ip} command must be available, which depending on distribution might not be installed by default.
