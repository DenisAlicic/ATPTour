Sends signal {\ttfamily signal} to all children processes of the process with pid {\ttfamily pid}, including {\ttfamily pid}. Signal defaults to {\ttfamily SIGTERM}.

For Linux, this uses {\ttfamily ps -\/o pid -\/-\/no-\/headers -\/-\/ppid PID} to find the parent pids of {\ttfamily PID}.

For Darwin/\+OSX, this uses {\ttfamily pgrep -\/P PID} to find the parent pids of {\ttfamily PID}.

For Windows, this uses `\textquotesingle{}taskkill /pid PID /T /F'\`{} to kill the process tree. Note that on Windows, sending the different kinds of POSIX signals is not possible.\hypertarget{md__home_denis_Documents_Faks_master_RS2_TennisAssociation_TennisAssociation_Client_node_modules_tree_kill_README_autotoc_md28587}{}\doxysection{Install}\label{md__home_denis_Documents_Faks_master_RS2_TennisAssociation_TennisAssociation_Client_node_modules_tree_kill_README_autotoc_md28587}
