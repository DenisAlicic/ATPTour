\begin{DoxyItemize}
\item {\bfseries{api\+:}} The API was significantly overhauled.
\item New protocol creating is now through protoduck.\+define() instead of protoduck()
\item Implementations are through Duck\+::impl instead of Duck(...)
\item The {\ttfamily private} option was removed
\item Static protocols were removed -- only method-\/style protocols are available now.
\item As part of that\+: the target argument to impl can no longer be omitted
\item The main export object is now the metaobject. protoduck.\+impl can be used to extend to MOP
\item .is\+Derivable is now a property on Duck instances, not a static method
\item .has\+Impl is now a method on Duck instances, not a static method
\item Protoduck will now genfunnify existing functions as default methods for genfuns declared in a protocol when implementing
\item Error messages have been overhauled to be more helpful
\item {\bfseries{deps\+:}} next\+Method is now an extra argument to methods
\item {\bfseries{meta\+:}} node$<$4 is no longer supported 
\end{DoxyItemize}
