By default, SMP measures loaders in groups. If truthy, this plugin will give per-\/loader timing information.

This flag is {\itshape experimental}. Some loaders will have inaccurate results\+:


\begin{DoxyItemize}
\item loaders using separate processes (e.\+g. {\ttfamily thread-\/loader})
\item loaders emitting file output (e.\+g. {\ttfamily file-\/loader})
\end{DoxyItemize}
