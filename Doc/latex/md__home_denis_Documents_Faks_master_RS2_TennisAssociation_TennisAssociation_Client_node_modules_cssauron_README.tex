{\ttfamily options} are an object hash of lookup type to string attribute or {\ttfamily function(node)} lookups for queried nodes. You only need to provide the configuration necessary for the selectors you\textquotesingle{}re planning on creating. (If you\textquotesingle{}re not going to use {\ttfamily \#id} lookups, there\textquotesingle{}s no need to provide the {\ttfamily id} lookup in your options.)

Returns false if the provided node does not match the selector. Returns truthy if the provided node {\itshape does} match. Exact return value is determined by the selector, based on the \href{http://dev.w3.org/csswg/selectors4/\#subject}{\texttt{ CSS4 subject selector spec}}\+: if only a single node is matched, only that node is returned. If multiple subjects are matched, a deduplicated array of those subjects are returned.
