{\bfseries{A Java\+Script Testing Framework}}

Jasmine is a Behavior Driven Development testing framework for Java\+Script. It does not rely on browsers, DOM, or any Java\+Script framework. Thus it\textquotesingle{}s suited for websites, \href{http://nodejs.org}{\texttt{ Node.\+js}} projects, or anywhere that Java\+Script can run.
