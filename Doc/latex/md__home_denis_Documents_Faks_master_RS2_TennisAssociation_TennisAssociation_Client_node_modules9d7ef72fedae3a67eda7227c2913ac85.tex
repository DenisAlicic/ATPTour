The default limit is the number of CPUs on the system -\/ 1, or 1.

The reason for subtracting one is that presumably the main thread is taking up a CPU as well, so let\textquotesingle{}s not be greedy.

Note that the array should be a list of Promise-\/\+\_\+returning\+\_\+ functions, not Promises themselves. If you have a bunch of Promises already, you\textquotesingle{}re best off just calling {\ttfamily Promise.\+all()}.

The functions in the queue are called without any arguments.
