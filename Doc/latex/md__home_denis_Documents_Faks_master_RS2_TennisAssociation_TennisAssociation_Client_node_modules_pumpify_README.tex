Combine an array of streams into a single duplex stream using \href{https://github.com/mafintosh/pump}{\texttt{ pump}} and \href{https://github.com/mafintosh/duplexify}{\texttt{ duplexify}}. If one of the streams closes/errors all streams in the pipeline will be destroyed.

Pass the streams you want to pipe together to pumpify {\ttfamily pipeline = pumpify(s1, s2, s3, ...)}. {\ttfamily pipeline} is a duplex stream that writes to the first streams and reads from the last one. Streams are piped together using \href{https://github.com/mafintosh/pump}{\texttt{ pump}} so if one of them closes all streams will be destroyed.
