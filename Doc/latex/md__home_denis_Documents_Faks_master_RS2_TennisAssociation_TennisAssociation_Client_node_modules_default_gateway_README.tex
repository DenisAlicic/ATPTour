Get the default network gateway, cross-\/platform. \end{quote}
Obtains the machine\textquotesingle{}s default gateway through {\ttfamily exec} calls to OS routing interfaces. On Linux and Android, the {\ttfamily ip} command must be available (usually provided by the {\ttfamily iproute2} package). \item {\ttfamily interface}\+: The name of the interface. On Windows, this is the network adapter name.

The {\ttfamily .v\{4,6\}()} methods return a Promise while the {\ttfamily .v\{4,6\}.sync()} variants will return the result synchronously.

The {\ttfamily gateway} property will always be defined on success, while {\ttfamily interface} can be {\ttfamily null} if it cannot be determined.
